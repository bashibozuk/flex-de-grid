# Flex Grid

Framework for building layouts using CSS flex API
## Installation

Install via npm:

```
$ npm install --save flex-de-grid
```

Install via bower:

```
$ bower install --save flex-de-grid
```

## Documentation
Documentation is at [docs/README.md](docs/README.md).
